# Interactive Quiz Website

This project is an interactive quiz website developed using React.js and styled with CSS.

## Overview

The Interactive Quiz Website offers engaging quizzes across various categories, providing users with an enjoyable learning experience.

## Features

- Interactive quizzes
- Diverse quiz categories
- User-friendly interface
- Responsive design

## Tech Stack

- React.js
- CSS

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/quiz-website.git`
2. Navigate to the project directory: `cd quiz-website`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open your browser and visit `http://localhost:3000` to view the app.
